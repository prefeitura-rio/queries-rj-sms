# populacao/

Define o universo de pacientes do monitoramento.

| Modelo | Papel |
|---|---|
| `int_monitora_cancer__populacao_alvo` | Universo inicial de pacientes do monitoramento (com gestante, contato e dados cadastrais). |
| `int_monitora_cancer__exclusoes` | Regras que filtram pacientes que devem sair da população-alvo (derivadas de eventos). |
